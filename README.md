## 📌 Project Overview

A complete data analysis project auditing a local café using 
real customer reviews from Google Maps.

The project answers 3 key business questions:
1. Where is the café losing customers?
2. What are competitors doing better?
3. What is the easiest way to increase sales?

## 🛠️ Tools Used

**| Tool | Purpose |**

| Microsoft Excel | Data collection and storage |
| SQL Server (SSMS) | Database creation and queries |
| Python (pandas, matplotlib) | Data cleaning and analysis |
| Power BI | Interactive dashboard |
| GitHub | Project sharing |

## 📊 Data Collected

**| Source | Data |**

| Google Maps | 30 customer reviews |
| Google Maps | Competitor ratings |

## 🗃️ Database Structure

**5 tables created in SQL Server**:

- **reviews** — 30 customer reviews with sentiment
- **business_info** — Café basic information
- **competitors** — 3 nearby competitor cafes
- **complaints** — 8 complaint categories
- **popular_items** — 7 most mentioned menu items

## 🔍 Key Findings

### 🚨 Top Problems:
1. **Hygiene Issues** — 4 reviews mention flies, mosquitos, uncovered food
2. **Slow Service** — 4 reviews mention long waiting for food and service
3. **Food Quality** — 3 reviews mention inconsistent quality

### ✅ Top Strengths:
1. **Sandwich** — Most loved item (5 positive mentions)
2. **Noodles** — Second most loved (4 positive mentions)
3. **Pocket friendly prices** — Appreciated by customers

### 🏆 Competitor Comparison:
**| Cafe | Rating | Reviews |**

| Scorpio Café | 4.0 | 2561 |
| Cafe Junk Delhi | 4.3 | 305 |
| Moustache Coffee | 3.8 | 427 |
| Veerji Malai Chaap | 3.9 | 214 |

## 💡 Recommendations

**| Priority | Problem | Action | Expected Impact |**

| 🚨 Urgent | Hygiene | Monthly pest control | Rating 4.0 → 4.3 |
| ⚠️ Important | Slow Service | Add evening staff | Rating 4.0 → 4.2 |
| ✅ Opportunity | Star Items | Promote on Zomato | Sales +20% |
